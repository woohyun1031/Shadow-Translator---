# Shadow Translator

영어 원문을 그대로 유지한 채 바로 아래에 `(한글 번역문)`을 함께 띄워주는 가벼운 확장 프로그램입니다.

영어 공부, 논문 리딩, 해외 직구를 할 때 구글 번역을 돌리면 원래 영단어가 뭐였는지 알 수 없어 답답하셨던 분들을 위한 필수 확장 프로그램입니다.

Shadow Translator는 크롬 브라우저 기본 번역 기능과 완벽하게 연동됩니다. 원문 텍스트를 통째로 한글로 덮어씌워버리는 기존 방식 대신, 원래의 영문 텍스트를 그대로 남겨두고 그 바로 아랫줄에 `(한국어 번역본)`을 자연스럽게 덧붙여 보여줍니다.

## 주요 기능

- **듀얼 자막 형식 제공**: 영어 원문과 한글 번역문을 상하로 동시에 배치하여 문맥을 정확하게 파악할 수 있습니다.
- **번역문 스타일링 적용**: 원문과 헷갈리지 않도록 번역문은 약간 작은 글씨의 회색 톤으로 은은하게 적용됩니다.
- **가장 가벼운 네이티브 호환성**: 무거운 외부 번역 API를 쓰지 않습니다. 크롬 브라우저 사용자라면 누구나 쓰는 내장 기본 번역기의 렌더링 과정을 역이용하여, 메모리 누수 없이 번개처럼 빠르게 동작합니다.

## 사용 방법

1. 브라우저 우측 상단 퍼즐 모양에서 **Shadow Translator**를 활성화합니다.
2. 해외 사이트(예: 아마존, 레딧, 영문 기사 등)에 접속합니다.
3. 빈 공간에 마우스 우클릭 후 [한국어로 번역]을 누르거나, 주소창 우측 G 모양 아이콘을 클릭해 번역을 실행하세요.
4. 원문이 번역문으로 싹 바뀌는 대신, 보기 좋게 둘 다 출력됩니다.

## 개인정보 보호
Shadow Translator는 외부 서버와 일절 통신하지 않으며, 사용자의 어떠한 인터넷 방문 기록이나 화면 내 텍스트 데이터를 외부로 수집 및 저장하지 않는 100% 안전하고 투명한 확장 프로그램입니다.

# Shadow Translator (English)

A productivity toolkit that displays both original and translated texts simultaneously when using Chrome's built-in translator.

Shadow Translator solves the frustration of original texts disappearing when you translate a webpage. It seamlessly integrates with Chrome's native translation engine to append the translated text directly below the original sentences. This optimizes your reading experience, making it perfect for language studying or reading foreign articles without losing context.

## Features of the Extension

- **Dual View Text**: When translating a page, Shadow Translator prevents the original text from being replaced. Instead, it neatly appends the `(translated text)` under the original sentence.
- **Clean & Minimal Styling**: To prevent visual clutter, the appended translation is styled with a subtly smaller font and a soft gray color, keeping your primary focus on the original text.
- **Lightning Fast & Native**: Shadow Translator uses ZERO external APIs. It directly hooks into Chrome's native rendering process, ensuring zero-delay performance, no memory leaks, and absolute privacy.
